# Resize me
This is an image resize and compression tool which can easily resize your image based on width, height or percentage.

## Demo
![alt text](https://github.com/Hakky54/resize-me/blob/master/images/demo.gif?raw=true)

### Requirements:
- Java 17
- Maven

### How to start
1. `mvn clean verify`
2. `mvn exec:java`
